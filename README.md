# Explainer Pages Repository

This repository contains a collection of **Explainer Pages** created by Michael Widener II.

Explainer Pages are simple, transparent documents designed to clarify the relationships between brands, projects, and organizations on the web.

They serve as reference pages that explain:

- ownership
- historical development
- brand relationships
- project context
- public-facing clarifications

These pages are intentionally simple HTML documents so they can remain readable and accessible long-term.
