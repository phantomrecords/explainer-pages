# Security Policy

## Overview

This repository contains **Explainer Pages**, which are simple HTML documents intended to provide transparent, factual explanations of organizations, brands, and projects.

Because this repository primarily contains static documentation and does not include application code, executable scripts, or backend systems, security risks are minimal.

However, maintaining the integrity and accuracy of the information in this repository remains important.

---

## Reporting a Concern

If you discover a security concern related to this repository, such as:

- malicious code inserted into a page
- unauthorized modification of files
- harmful external links
- content that could compromise user safety

please report it to the repository maintainer.

### Contact

Michael Widener II  
https://michaelwidener.com/

Please include:

- the file affected
- a description of the issue
- steps to reproduce the concern if applicable

---

## Responsible Disclosure

If a concern is reported, the maintainer will review the issue and take appropriate action, which may include:

- removing or correcting affected content
- updating repository policies
- documenting the change in the repository history

---

## Scope

This security policy applies to the contents of the **Explainer Pages repository only**.

External websites referenced within explainer pages are maintained independently and are outside the control of this repository.

---

## Good Faith

This repository is maintained in good faith to provide transparent, factual documentation. Any reported concerns will be reviewed with the goal of maintaining accuracy, integrity, and public trust.
