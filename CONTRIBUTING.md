# Contributing

Explainer pages in this repository are maintained by Michael Widener II.

These documents are intended to be historically accurate summaries of brands, projects, and organizations.

If corrections are needed, please open a pull request or submit a documented correction with verifiable sources.

Contributions should:

• Maintain factual accuracy  
• Preserve a neutral explanatory tone  
• Avoid promotional or marketing language  

Thank you for helping keep this archive accurate.
